# Playable Apollo
Adds all combat arts to Apollo's player file with proper animation support, does not touch PVP Apollo.
<img width="314" height="284" alt="myvideo-1-ezgif com-video-to-gif-converter" src="https://github.com/user-attachments/assets/9b421289-98b0-4506-869b-73d91470a729" />


### To play as Apollo:
This mod does not contain a method to change characters on its own. Use the **char-select** mod.
- https://github.com/CCDirectLink/char-select

### Affects Apollo as a party member:
Apollo as a party member will have **all** the level 1 combat arts available to use, not just the limited moveset he has in vanilla. Additionally, if you install the **cc-combat-art-level-extender** mod, he will also be able to use level 2 and 3 combat arts as a party member.
- https://github.com/CCDirectLink/cc-combat-art-level-extender/releases


### Install Insructions:
Install from the in-game mod browser.

### Dependency Mods:
None!
